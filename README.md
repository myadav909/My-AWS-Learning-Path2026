# AWS Cloud Journey 🚀
## Section 1: IAM (Identity and Access Management)
* IAM is **global**, not regional.
* Best Practice: Never use the root account for daily tasks; create an IAM User instead.
## AWS Console Home Overview - L12
# Regions Selector: 
Choose regions to reduce latency; important for service availability.
# Console Layout: 
Displays recently used services, AWS account usage, and available tutorials.
# Searching Services: 
Use the search bar to quickly find services like Route 53 without memorizing all AWS services.
# Route 53: 
A globally available service; does not require region selection unlike other AWS services (e.g., EC2).
# AWS Global Infrastructure: 
Familiarize yourself with AWS regions and their available services for effective cloud resource management.
